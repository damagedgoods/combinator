How to

Preparing your CSV file

It's super simple. You just need to create a spreadsheet (Excel, Numbers, Google Sheets or whatever) and add a column of values for each part.

For example, if you want a Cluedo generator, you will need to create a column for the person, another for the place and another for the weapon. Just populate each column with values.

Once all your data is ready, make sure you export it as CSV. This part is important!

You can also download a CSV sample file and just change the values.
