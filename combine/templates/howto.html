How to

Preparing your CSV file

It's super simple.

  1. First, create a new spreadsheet (Excel, Numbers, Google Sheets or whatever).
  2. Use a column for each part of the mix. For example, if you want a Cluedo generator, you will need to create a column for the person, another for the place and another for the weapon.
  3. Populate each column with values. The more the funnier.
  4. Once all your data is ready, export it as CSV and use it to create a new mix.

You can also download a CSV sample file and just change the values.
