About the MixMaster

What is the MixMaster?

The MixMaster* helps you to create new concepts, sentences and ideas, randomly combining previously defined parts. It might be useful for brainstorming sessions, inspiration processes, cocreation workshops or just to have fun.

How it started?

It started as an excuse to learn how to build a full website from scratch, including design, front and back end programming. The source code is available. If you find any bug or have a great idea to make ir better, feel free to add them as a Github Issue.

Thanks for using it!
